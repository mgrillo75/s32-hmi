  WHAT THE VERIFICATION SCRIPT TESTS
═══════════════════════════════════════════════════════════════════════════════

1. COILS (Circuit Breaker Status)
   ─────────────────────────────────
   Reads: Coils 0, 1, 2
   Tests: Gen_1_52A, Gen_2_52A, Gen_3_52A
   
   Output shows:
   • Raw boolean values [true/false, true/false, true/false]
   • Decoded status: "CLOSED ✓" or "OPEN ✗" for each breaker

2. HOLDING REGISTERS (Generator Power)
   ──────────────────────────────────────
   Reads: Holding registers 0-3 (first 4 registers)
   Tests: GenSet_1 (0-1), GenSet_2 (2-3)
   
   Output shows:
   • Raw 16-bit register values
   • Decoded 32-bit unsigned values (MSR byte order)
   • Hex representation
   • Possible scaled values (if > 1000, shows ÷100 for kW)

═══════════════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

If coil read fails:
  • Coils may not be configured on your device
  • Check that coil addresses 0-2 exist
  • Verify device supports function code 1 (Read Coils)

If holding register read fails:
  • Check that holding registers 0-3 exist on device
  • Verify device supports function code 3 (Read Holding Registers)
  • Ensure addresses match your device's register map

If connection fails:
  • Verify slaveHost and slavePort in modbus.config.json
  • Check that device is powered on and accessible
  • Test network connectivity: ping 127.0.0.1
  • Verify unitId matches device configuration
